# Plasgos CRM Landing Page

## Cara Menjalankan Project

1. Clone repository

```bash
git clone <repository-url>
cd <project-folder>
```

2. Install dependencies

```bash
npm install
```

3. Jalankan development server

```bash
npm run dev
```

4. Buka browser dan akses `http://localhost:5173`
